## 🌟 Project Overview

**PyClimaExplorer** is a premium, highly interactive web application designed for researchers, scientists, policy makers, and the general public to explore complex climate datasets (CESM, ERA5, CMIP6) with intuitive and aesthetic visualizations. By translating complex `.nc` files into breathtaking visual narratives, PyClimaExplorer acts as your command intelligence center for planetary climate telemetry.

Developed for the **TECHNEX '26 Hackathon**.

## 🚀 Key Features

*   **Scientific Command Center**: 9 dynamic ECharts providing multi-dimensional insights (Zonal, Meridional, Statistical, Radar, Heatmap).
*   **NetCDF File Port**: Upload and analyze your own `.nc` (Network Common Data Form) files directly through the dashboard.
*   **Geospatial Intelligence Map**: Plotly-powered interactive 2D maps and 3D topographical globes with multiple rendering modes.
*   **AI Command Analysis**: Integrated machine learning modules to generate predictive projections and intelligence briefings.
*   **Orbital Data Acquisition**: Direct integration with ECMWF ERA5 via Copernicus CDS API.
*   **Real-World Engine**: Robust support for Dask (large files) and `cftime` (non-standard climate calendars).

## 📖 Overview

**PyClimaExplorer** is a cutting-edge, rapid-prototype interactive visualizer designed to turn raw atmospheric numbers into a compelling story about our changing planet. Born out of the **Technex '26 Hackathon**, this platform provides an intuitive interface to slice, dice, and visualize complex multi-dimensional climate data in real-time.

> [!IMPORTANT]
> This project was developed within a rigorous **24-hour hackathon** window, successfully meeting all core deliverables and implementing multiple bonus features including AI-driven analysis and 3D geospatial rendering.

---

## 🚀 Key Intelligent Modules

### 🛰️ Spatial Intelligence Heatmap
Visualize global climate variables (Temperature, Precipitation, etc.) with high-fidelity heatmaps. Toggle between 2D distributions and 3D intensity rendering to identify climate anomalies.

### 🌍 Planetary Navigation (3D Globe)
A high-tier bonus feature providing a spatial distribution view that renders climate intensity directly onto a 3D Earth model. Supports interactive rotation and real-time data mapping.

### 📈 Temporal Analytics & ML Forecasting
Track climate trends over time with interactive time-series plots. Includes moving averages, anomaly detection, and stochastic projections for future climatic states.

## 🛠️ Technical Architecture

| Layer | Technology |
| :--- | :--- |
| **Frontend UI** | Streamlit (Custom Glassmorphism CSS) |
| **Data Orchestration** | Xarray, Dask (for large-scale arrays) |
| **Intelligence Engine** | Groq AI (Llama 3), NumPy, Scipy |
| **Visual Library** | Plotly, Apache ECharts, PyDeck |
| **Climate Science** | NASA MERRA-2 Global Diagnostics |

---

## 🛡️ Data Authenticity
This system is powered by the **NASA MERRA-2 (Global Diagnostics)** dataset, sourced from the Goddard Earth Observing System (GEOS). Every visualization is a reflection of verified scientific telemetry.

## ⚙️ Installation & Local Setup

### 1. Prerequisites
- Python 3.9+
- A Groq API Key (for AI Analyst features)

### 2. Quick Start
```bash
# Clone the repository
git clone <repository-url>
cd Py-Climax

# Setup environment
python -m venv venv
venv\Scripts\activate  # On Windows

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo GROQ_API_KEY=your_key_here > .env

# Run the platform
streamlit run app.py
```
